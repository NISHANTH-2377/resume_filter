before running the program please extract the datasets_&_candidates.rar to the resume_filter folder as it contains the datasets for training and candidates list for inference.

if you want to test the model , first extract the datasets_&_candidates.rar file, then replace the candidates.jsonl file with yours and run the test_model.py

the output will be produced as a .csv file which can be found in the 'output' directory.

the train_model.py is the program used to train the model.

again it requires the candidates.jsonl file to be present in the 'datasets' directory.
